# เอกสารขอบเขตโครงการ (Scope Document)
**Project Name:** BMI Tracker Firebase
**Technology Stack:** Web Application (HTML/CSS/JS) + Firebase

## 1. วัตถุประสงค์ (Objective)
พัฒนาเว็บแอปพลิเคชันมีระบบ Login สามารถเก็บข้อมูลผู้ใช้ สำหรับคำนวณค่าดัชนีมวลกาย (BMI) ที่สามารถบันทึกประวัติและแสดงแนวโน้มสุขภาพของผู้ใช้งานได้ เพื่อช่วยให้ผู้ใช้ตระหนักถึงสถานะสุขภาพเบื้องต้น

## 2. ขอบเขตการทำงาน (IN Scope)
* **User Authentication :** รองรับการเข้าสู่ระบบ (Sign-in) เพื่อแยกข้อมูลประวัติของผู้ใช้แต่ละคน
* **BMI Calculation :** คำนวณค่า BMI และ ระบุเกณฑ์สุขภาพ (Underweight, Normal, Overweight, Obese) ได้ถูกต้อง
* **Data History :** แสดงรายการประวัติย้อนหลังของผู้ใช้ โดยเรียงจากวันที่ล่าสุด
* **Input Validation :** ป้องกันการกรอกข้อมูลที่ผิดรูปแบบ (เช่น ค่าติดลบ, ตัวอักษร, หรือค่าที่เป็นไปไม่ได้ เช่น สูง 300 ซม.)
* **Recommended Exercises :** แสดงข้อความแนะนำการออกกำลังกายและการปฏิบัติตัวเบื้องต้น โดยอ้างอิงจากเกณฑ์ BMI ที่คำนวณได้ 

### รายละเอียดฟังก์ชัน (Checklist)
* **การรับข้อมูล (Input):**
    * [x] รับค่า น้ำหนัก (กิโลกรัม) และ ส่วนสูง (เซนติเมตร) จากผู้ใช้
    * [x] มีการตรวจสอบความถูกต้อง (Validation) เช่น ห้ามใส่ค่าติดลบ หรือตัวหนังสือ
* **การประมวลผล (Process):**
    * [x] คำนวณค่า BMI ตามสูตรมาตรฐาน
    * [x] แปลผลค่า BMI เป็นเกณฑ์ (เช่น ผอม, ปกติ, ท้วม, อ้วน)
    * [x] เลือกคำแนะนำการออกกำลังกายที่เหมาะสมตามเกณฑ์ BMI (Logic แบบ If-Else)
* **การจัดเก็บข้อมูล (Data Storage - Firebase):**
    * [x] บันทึกวันที่, น้ำหนัก, ส่วนสูง, และค่า BMI ลงใน Firebase Firestore
* **การแสดงผล (Output/Reporting):**
    * [x] แสดงค่า BMI และเกณฑ์แปลผล
    * [x] แสดงประวัติการบันทึกย้อนหลังในรูปแบบ ตารางรายการ (Table List) เรียงตามวันที่ล่าสุด
    * [x] แสดงข้อความแนะนำสุขภาพและการออกกำลังกาย

## 3. สิ่งที่อยู่นอกขอบเขต (OUT Scope)

* [x] ไม่มีการแนะนำเมนูอาหารหรือการออกกำลังกายแบบเจาะลึก
